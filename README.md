# Multi-Branch Recruitment & Applicant Tracking System (ATS)

## Project Description
A full-stack web application that automates the hiring process for a software company operating across multiple branches (Islamabad, Lahore, Karachi, Remote).

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MongoDB Atlas
- File Storage: Cloudinary
- Email: Gmail SMTP
- Deployment: Vercel + Railway

## Setup Instructions
### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
